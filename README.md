# 동행 3D 시뮬레이터

## 프로젝트 소개
이 프로젝트는 이동약자가 학교 앞 횡단보도 또는 보도 환경에서 겪는 이동 장벽을 3D로 체험할 수 있도록 만든 웹 시뮬레이터입니다.

## 이동약자 유형
- 휠체어 사용자
- 고령 보행자
- 시각장애인 중 우리 팀이 선택한 유형

## 주요 장면
1. 출발 지점
2. 장애물 구간
3. 우회 경로
4. 도착 지점
5. 결과 화면

## 조작 방법
- W: 앞으로 이동
- A: 왼쪽 이동
- S: 뒤로 이동
- D: 오른쪽 이동
- 마우스: 시점 이동

## 체험 포인트
- 어떤 장애물에서 이동이 어려웠는가?
- 이동이 막힌 이유는 사람의 문제가 아니라 환경 설계의 문제인가?
- 하나만 개선한다면 어떤 공간을 바꾸고 싶은가?
