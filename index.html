    이동 보조 앱이 왜 필요한지 직접 확인합니다.

    같은 길을 이동약자 유형별로 체험하고, 앱이 제공해야 할 거리·방향·신호·위험·행동 정보를 발견해 보세요.

    시뮬레이터와 프로젝트의 역할

    3D 시뮬레이터는 이동 장벽과 정보 부족을 확인하는 사전 체험 환경입니다. 학생 프로젝트의 최종 목표는 이동약자의 안전한 이동을 돕는 앱을 설계·개발하는 것입니다.

    어떤 이동 조건으로 체험할까요?

    장애 정도를 평가하는 화면이 아닙니다. 카메라 높이·속도·폭·정보 접근·횡단 조건의 차이를 비교합니다.

    앱 도움 조건

    이동 보조 앱의 도움을 받을까요?

    첫 체험은 앱 도움 없이 진행한 뒤, 결과 화면에서 같은 조건으로 앱 도움을 켜고 다시 체험하는 것을 권장합니다.

        도움말

        연속 체험과 이동 보조 앱 관찰 방법

        • 연속 이동: 장애물마다 정답 선택 창이 열리지 않습니다. 대기·출발·우회·탐지 같은 실제 행동으로 기록합니다.
        • 신호: 적색에는 대기하고, 녹색과 남은 시간을 확인해 출발합니다. 종료 임박에는 차량이 계속 정지합니다.
        • 앱 OFF/ON: OFF에서는 환경 정보만 사용하고, ON에서는 거리·방향·신호·예상 시간·권장 행동을 안내합니다.
        • 시각장애 모드: F로 지팡이 탐지를 사용합니다. 앱 OFF에서는 음향신호기 유무에 따라 신호 정보 접근이 달라집니다.
        • 분석: 강사 분석 패널에서 상황인지·판단·동작수행과 앱 입력→판단→출력 근거를 확인할 수 있습니다.
